When the notifications are used?
Notifications area created by the application to inform you if a special event occurs. Notifications are restricted only for contributors. The notifications can be viewed by accessing Notifications tab from the side menu.
1. Notifications for category limit events
These notifications are created when a category monthly limit is about to be reached (yellow notifications), or is exceeded (red notifications).
2. Notifications for loans
These notifications are created when Until date for a loan is exceeded, informing you that you have to receive or to give a loan to a person.
