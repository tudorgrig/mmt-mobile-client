How to create a new loan?
1. Choose "Loans" from the side menu
2. Click "Add loan" from bottom of screen
3. Loans section is available only for contributors. Fill the fields needed to create a loan.
Select a person from the list or choose Add new person? if is the first time you insert a loan for this person.
Choose Are you receiving? if you receive the money from this person, or Are you giving? if you give the money to the person.
4. Press "Create loan".
That will create the loan and return you to the loan list page.
You can also press "Create & New". That will create the loan and let you create another loan.
