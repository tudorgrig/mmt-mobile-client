How to create a new category?
1. Choose "Categories" from the side menu
2. Click "Add category" from bottom of screen
3. Fill the fields needed to create a category. Limit field allows you to set a monthly expense limit for this category. Setting monthly limit for categories is permited only for contributors
4. Press "Create category".
That will create the category and return you to the category list page.
You can also press "Create category & New". That will create the category and let you create another category.
